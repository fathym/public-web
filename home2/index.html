Say goodbye to unwieldy frontend monoliths.

Fathym automates builds, hosting and deployment so you can focus on building Jamstack sites or Single Page Applications with the JavaScript frameworks and Static Site Generators you love. Steer clear of unwieldly frontend monoliths. Break your website down into smaller, flexible and more manageable pieces for independent builds and deployments.
TAKE THE FATHYM JOURNEY

1. Start with your code

Getting started is as simple as connecting your GitHub repository. Select an open-source recipe from your favourite frameworks and Static Site Builders to see Fathym in action or bring in your own code.

Your code

Bring your sites and applications that you already have in GitHub or NPM.

Build and deploy

Let us handle the automated build pipelines and deployments behind the scenes so you can stick to the good stuff.

2. Break it down

Break website projects down into smaller apps with individual repositories and automated build pipelines for:
  1. smaller, more cohesive, decoupled codebases
  2. smaller, faster builds
  3. simple testing, version control, release management
  4. incremental upgrades and independent deployments

You're a composer

Build with the freedom to use different JavaScript frameworks or Static Site Generators on the one site. Use the right framework for the right job. Fathym effortlessly brings your project’s apps together to compose one cohesive website or web app on your custom domain.

3. Deploy with confidence

The Fathym Global Edge Network delivers your content and sites at global points closest to your users.
Update to any version of your site through our version control, and easily rollback when things go awry.

4. Scale as you grow

Get started for free in a shared infrastructure environment built to scale with you and your customer's needs.
Outgrown Fathym's shared infrastructure? Employ all the same great tools from your own private or managed clouds.

Why deploy with Fathym?

Fathym's powerful micro frontends allow work to be divided and conquered concurrently. Virtual developers handle the heavy lifting of the hosting and DevOps automation, saving both time and money. Plus, the ability to use multiple frameworks! Discover new ways to maximize resources and knowledge from your team with your code, low code, and no code tools. Harness solutions from the open source universe or developed by your team.
