Say goodbye to unwieldy frontend monoliths.

Fathym automates CI/CD builds, deployment and hosting so you can focus on building Jamstack sites or Single Page Applications with the JavaScript frameworks, Static Site Generators or no code builders you love. Steer clear of clunky frontend monoliths. Break your web project down into smaller and more manageable pieces for independent, flexible and scalable deployments.
TAKE THE FATHYM JOURNEY

1. Start with your code.

Getting started is as simple as connecting your GitHub repository. Select an open-source recipe from your favorite frameworks or static site generators see Fathym in action or bring in your own code.

Your code.

Bring your sites and applications that you already have in GitHub or NPM.

Build and deploy.

Let us handle the automated build pipelines and deployments behind the scenes so you can stick to the good stuff.

2. Break it down.

Break website projects down into modular, independently versioned apps with individual code repositories and automated build pipelines for:
  1. Smaller, more cohesive, decoupled codebases.
  2. Smaller, faster builds.
  3. Simple testing, version control and release management.
  4. Incremental upgrades and independent deployments.
  5. Flexibility to fork and re-use modular apps for other projects.

You're a composer.

Build with the freedom to use different JavaScript frameworks, Static Site Generators or even no code builders on the one domain, if that works for you. Use the best tool for the job. Fathym effortlessly brings different repos and stacks together to compose one cohesive and seamless website or web app on your custom domain.

3. Deploy with confidence.

The Fathym Global Edge Network leverages Azure CDN to deliver your content and sites to the global points closest to your users. Take advantage of independent builds, versions and deployments to rapidly release changes, upgrades, rollbacks and fixes.

4. Scale as you grow.

Get started for free in a shared infrastructure environment built to scale with you and your team's needs. Outgrown Fathym's shared infrastructure? Employ all the same great tools from your own automated private Azure cloud.
Take advantage of our simplified developer experience while scaling to your own Azure cloud with full access to your infrastructure and the full range and power of Azure's extensive services. Build a future-proof tech stack from the get go without the complexity.

Stay in sync with your team.

Collaborating and keeping updated with your teammate’s builds, pull requests and deploys has never been easier than through our social feed. Fathym offers one consolidated workflow for you to view and manage your web projects and their corresponding repositories and builds.

Leverage the power of Azure minus the complexity.

Fathym leverages Azure for the most advanced, enterprise-grade infrastructure and world class performance, reliability and security. Control who accesses your apps through out-of-the-box authentication and identity management. As you scale and need the flexibility and full range of services that only the big cloud providers offer, we'll automate best practice infrastructure in your own Azure environment.

Why deploy with Fathym?

Fathym deploys, hosts and integrates multiple repos and your favourite tech stacks, frameworks, no code builders and data sources together on one domain. Take advantage of Fathym's streamlined PaaS developer experience while future-proofing your tech stack knowing that you can scale to your own automated Azure cloud when ready.
